<h2 align="center">ğŸ“– Usage</h2>

### Browsing Wallpapers

1. Launch the application
2. The **Top** tab shows trending workshop wallpapers
3. Use the dropdown menus to change:
   - **Sort Method**: Most Popular, Most Recent, Last Updated, Most Subscribed
   - **Time Period**: All Time, Today, Week, Month, etc.
4. Use **Prev/Next** buttons to navigate pages

### Searching

- **By Keywords**: Type in the search box and press Enter or click "Search"
- **By Workshop ID**: Paste a Workshop ID (e.g., `3309647577`) directly

### Downloading

1. Find a wallpaper you like
2. Click the **Download** button
3. Wait for the download to complete
4. The wallpaper is automatically saved to your Wallpaper Engine folder

### Managing Downloads

1. Switch to the **Downloaded** tab
2. View all your downloaded wallpapers
3. Click **Open folder** to view files in Explorer
4. Click **Remove** to delete a wallpaper
